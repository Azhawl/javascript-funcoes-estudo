# JavaScript - Estudo de Funções

Este projeto contém exemplos e exercícios práticos sobre funções em JavaScript.

## Estrutura do Projeto

```
javascript-funcoes-estudo
│
├── README.md
├── package.json
│
├── src
│   ├── 01-funcoes-basicas.js
│   ├── 02-parametros-retorno.js
│   ├── 03-arrow-functions.js
│   ├── 04-escopo.js
│   ├── 05-funcoes-anonimas-callback.js
│   ├── 06-funcoes-de-ordem-superior.js
│   ├── 07-closures.js
│   └── 08-desafios.js
│
└── exercicios
    ├── nivel-1.js
    ├── nivel-2.js
    └── nivel-3.js
```

## Conteúdo

### src/
Contém exemplos práticos e explicações sobre diferentes aspectos de funções em JavaScript:
- **01-funcoes-basicas.js**: Introdução às funções
- **02-parametros-retorno.js**: Parâmetros e valores de retorno
- **03-arrow-functions.js**: Arrow functions (ES6)
- **04-escopo.js**: Escopo de variáveis e funções
- **05-funcoes-anonimas-callback.js**: Funções anônimas e callbacks
- **06-funcoes-de-ordem-superior.js**: Higher-order functions
- **07-closures.js**: Closures e encapsulamento
- **08-desafios.js**: Desafios práticos

### exercicios/
Contém exercícios práticos organizados por nível de dificuldade:
- **nivel-1.js**: Exercícios básicos
- **nivel-2.js**: Exercícios intermediários
- **nivel-3.js**: Exercícios avançados

## Como usar

1. Navegue até o diretório do projeto
2. Execute os arquivos com Node.js:
   ```bash
   node src/01-funcoes-basicas.js
   ```
3. Complete os exercícios em `exercicios/`

## Requisitos

- Node.js (versão 14 ou superior)
